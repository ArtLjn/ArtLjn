我是 **ArtLjn (LJN)**，一名深耕 **AI Agent 生态** 与 **开发者工具链** 的全栈工程师。热衷于探索 **MCP (Model Context Protocol)** 的无限可能，致力于构建连接 AI 逻辑与现实业务的桥梁。

<div align="center">

| 原则 | 释义 |
|:---:|:---|
| **简洁胜过复杂** | 即使是复杂的 Agent 逻辑，也要有优雅的架构 |
| **自动化即生命** | 如果一件事要重复三次，那就写个工具（比如 Ocean 系列） |
| **拥抱开源生态** | 从 MCP 到本地大模型，在开源的浪潮中保持进化 |
